# Telethon Contact Exporter (Vercel Version)

Export kontak Telegram via API:
- `/api/generate-contacts?page=1` ➔ Tampilkan kontak per 100 per halaman
- `/api/download-contacts` ➔ Download semua kontak sebagai file .txt

## Deployment
- Fork repo ini
- Connect ke Vercel
- Done!

## Requirements
- fastapi
- telethon
- phonenumbers
- uvicorn
